OpenStreetMap

OpenStreetMap (OSM) is a community of 2m volunteers adding data to an open global vector map

sometimes the aims are humanitarian

the humanitarian OSM team, HOT OSM, has recently been mapping Vanuatu due to the major storm that hit there

one way that users contribute to OSM is through tracking their own movements with GPS

and exporting the GPX file

that’s what we’ll do today

this is geoJSON - it’s a cool vector format

that was being presented on github where you’ll need to make an account for this lab

finally!

generic steps for this lab:

  • digitize a line
  • look at it in your phone
  • start GPS logging
  • walk that line
  • stop GPS logging
  • upload both digitized line and logged GPX to geojson.io
  • save as a GitHub gist and share publically
